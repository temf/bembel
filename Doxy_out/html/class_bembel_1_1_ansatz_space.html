Detailed Description

template<typename Derived>
class Bembel::AnsatzSpace< Derived >

The AnsatzSpace is the class that handles the assembly of the discrete basis. 

It invokes a superspace and uses the Glue and Projektor class to

assemble a transformation matrix, which relates the superspace to the desired basis.
