Detailed Description

template<int N = 10>
class Bembel::IO::Logger< N >

A simple class for producing output.

A simple class for producing output. Its not meant as a logger in the classical sense, but rather to write measurement data in a machine & human readable output. For example, it can be used to generate files to be read by pgfplots.
