This example computes the solution of a Laplace problem.

Consider the following Dirichlet problem. Let \(\Omega\) be a compact domain with boundary \(\Gamma\). Given a function \(g\) on \(\Gamma\), find a function \(u\) on \(\Omega\) such that,

\begin{aligned} \Delta u &= 0\quad\mathrm{in}\, \Omega \\ u &= f\quad\mathrm{on}\, \Gamma. \end{aligned}

The solution can be computed with the representation formulae

\begin{aligned} u(x) = \tilde V(\omega)(x)\quad\mathrm{in}\,\Omega, \end{aligned}

where

\begin{aligned} \tilde V(\mu)(x) = \int_\Gamma \frac{\mu(y)}{4\pi\|x - y\|}\,\mathrm{d}y \end{aligned}

is the Laplace single layer potential with some density \(\mu\). Utilizing the trace operator \(\gamma\), the single layer operator is defined by \(V = \gamma\circ\tilde V\). When discretized in a conformal finite-dimensional function space \(\mathbb{S}\), the variational formulation is as follows: Find \(\omega\in\mathbb{S}\), such that

\begin{aligned} \int_\Gamma \mu \int_\Gamma V(\omega)(x)\,\mathrm{d}x = \int_\Gamma \mu f\,\mathrm{d}x\quad\forall\mu\in \mathbb{S} \\ \end{aligned}

holds, which is realized in this example.
